# Terraform: Terraria-Style game

<img width="991" height="463" alt="image" src="https://github.com/user-attachments/assets/bf31c993-c199-4d6b-8c5e-76ce8b93a801" />
<p>An attempt to make basic terrain generation and movement similar to that of terraria.</p>
<p>This showcases terrain generation using perlin noise movement, and physics.</p>

# Get started

<p>Run this to clone the repository:</p>

```bash
git clone https://github.com/sarbarthasengupta2012-prog/terraria-style-sfml-game.git
cd terraria-style-sfml-game
```

<p>Run main.cpp (YOU'LL NEED A COMPILER LIKE GCC OR CLANG, THIS COMMAND IS FOR A GCC COMPILER)</p>

```bash
g++ main.cpp -o game
```

# Project Features
<ul>
  <li>Terrain generation using perlin noise</li>
  <li>WASD Movement</li>
  <li>Slide mechanic (when you hit the ground while running left or right then stop just as you hit the ground, you do a little slide)</li>
  <li>Physics and velocity</li>
</ul>
